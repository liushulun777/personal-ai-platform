# 环境切换配置说明

## 一键切换

修改 `config/application-env.yml` 文件中的 `INFRA_HOST` 值即可切换所有服务的运行环境：

```yaml
# 公司环境
INFRA_HOST: 172.18.0.34

# 家里环境
INFRA_HOST: 192.168.5.2
```

**切换后重启所有服务即可生效。**
